## 🔄 **API Integration**

### **Connected Endpoints:**
All API calls connect to your live AWS backend:
```
https://afbtrc48hc.execute-api.us-west-2.amazonaws.com/development
```

### **API Operations:**
- ✅ `POST /auth/login` - User login
- ✅ `POST /families` - Create family
- ✅ `POST /children` - Add child
- ✅ `GET /children` - List children
- ✅ `POST /expenses` - Add expense
- ✅ `GET /expenses` - List expenses
- ✅ `POST /funds` - Add funds

All authenticated with JWT tokens stored in LocalStorage.
